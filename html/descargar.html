Linux

Sistemas basados en debian

1: Ejecuta el siguiente comando en la terminal

sudo apt install quasiestado

Sistemas basados en redhad

1: Ejecuta el siguiente comando en la terminal

sudo dnf install quasiestado

2: comprueba que los paquetes se haya installado de forma correcta

Listo: ya tienes Quasiestado instalado en tu ordenador

done
Puede que quasiestado no funcione con algunas destribuciones, para esto busca en nustro foro oficial
